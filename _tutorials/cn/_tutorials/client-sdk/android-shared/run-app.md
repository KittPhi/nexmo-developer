运行应用
====

您可以在物理电话上启动应用（[USB 调试功能已启用](https://developer.android.com/studio/debug/dev-options#enable)），也可以创建新的 [Android 虚拟设备](https://developer.android.com/studio/run/managing-avds)。设备出现时，按 `Run` 按钮：
