运行您的应用程序
========

在浏览器中打开 `index.html`。

页面上会显示文本字段和按钮。

在文本字段中输入 [E.164](/concepts/guides/glossary#e-164-format) 格式的手机号码（例如，`447700900000`），然后点击 `Call` 按钮。

您还可以打开 Web 浏览器控制台，在控制台查看 Web 应用和 Client SDK 的日志消息。您将看到该应用程序尝试连接到您提供的目标号码。

一旦呼叫接通，您可以应答呼叫，然后听到应用内的语音通话。
